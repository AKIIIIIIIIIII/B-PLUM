# b-plum Portfolio

A Nordic-style minimalist portfolio landing page.

## Tech Stack
- HTML5
- Tailwind CSS (via CDN)
- Google Fonts (Inter, Playfair Display, Noto Sans JP)

## Deployment
This project is ready to be deployed to Netlify. Just upload the `index.html` file or connect your repository.
